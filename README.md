《面具英雄》是一款基于现实世界的共创故事游戏，
玩家们将扮演被称为“面具英雄”的超级英雄，和超凡存在签订契约，
以这样的身份在现实世界生活。

本游戏适合1-6 位玩家和一名主持人共同游玩。

## 玩家
如果您希望作为玩家游玩《面具英雄》，您需要：

- 寻找一名主持人来主持该游戏，并进行基础教学；
- 阅读核心规则中玩家手册中第三章《玩家手册》部分并创建角色卡；
- 阅读其他主持人希望您阅读的部分。

## 主持人
如果您希望主持一场《面具英雄》游戏：

- 请阅读核心规则中的所有内容；
